# 🚀 Liudup v3.0 - 一体化 AI 数据工厂 (Liudup)

**Liudup** 是一个专为广告素材素材管理、YOLO 模型训练设计的全链路数据闭环系统。在 v3.0 版本中，我们实现了从“数据去重”到“智能标注”，再到“可视化训练”与“云端发布”的完整生产力闭环。

---

## 🌟 一、 核心功能模块

### 1. 智能去重 (Cleaner) - [100% 完成]
*   **双引擎支持**：集成 MobileNetV2 (CNN) 高速去重与 DINOv2 (ViT) 深度语义理解。
*   **强一致性分组**：基于视觉指纹自动识别相似素材。
*   **质量优先清理**：智能对比分辨率与文件大小，一键保留最高质量原图。
*   **素材备份机制**：清理的重复素材自动归档至 `_backup` 目录，确保数据安全。

### 2. 交互标注 (Labeller) - [100% 完成]
*   **矢量标注画布**：基于 `QGraphicsView` 的高性能画布，支持无损缩放、平移与矩形框绘制。
*   **动态类别管理**：支持用户实时添加、删除、修改标签类别，自动同步至 `config.json`。
*   **🤖 AI 辅助标注**：一键加载训练产出的 `best.pt` 模型进行预识别，标注员仅需纠偏，效率提升 500%。
*   **YOLO 原生对接**：实时生成归一化坐标并保存为标准的 `.txt` 标注文件。

### 3. 训练工厂 (Trainer) - [100% 完成]
*   **全自动流水线**：一键完成数据集随机划分（Train/Val）、目录组织与 `data.yaml` 生成。
*   **可视化监控**：集成 `pyqtgraph` 实时解析训练指标，动态绘制 Loss 曲线与 mAP 精度图。
*   **异步训练执行**：封装 `ultralytics` YOLOv8 后台线程，训练过程中 UI 保持丝滑流畅。

### 4. 发布中心 (Publisher) - [100% 完成]
*   **Minio 深度集成**：支持从远程桶拉取素材并同步清理。
*   **一键云端发布**：训练完成后，一键将带时间戳的最佳模型（best.pt）发布至 Minio 指定路径。

## 🏗 二、 技术架构特性

1.  **模块化工作区 (Workspace)**：
    *   采用 **侧边导航栏 + QStackedWidget** 架构，业务逻辑在 `CleanerPage`、`LabellerPage`、`TrainerPage` 之间完全解耦。
2.  **数据安全与隔离 (Data Isolation)**：
    *   **独立索引表**：去重特征存储在 `file_mapping`，标注数据存储在 `label_records`。
    *   **抗风险重算**：全量重算去重特征时，绝对不会触及已经完成的标注成果。
3.  **多线程性能优化**：
    *   所有重量级任务（特征提取、Minio 同步、YOLO 训练）均在独立 `QThread` 中运行，并提供 ETA 时间平滑算法。

## 👨‍💻 四、 开发者指南

### 环境依赖
确保安装以下库以支持 v3.0 的所有特性：
```bash
pip install torch torchvision numpy pillow PyQt5 boto3 ultralytics pandas pyqtgraph python-dotenv
```
# 手机端导出必备
```bash
pip install onnx onnxslim  # ONNX 导出
pip install tensorflow     # TFLite 导出（包比较大，如果只要 ONNX 可以不装）
```
### 快速启动
1. 配置 `.env` 文件中的 Minio 连接信息。
2. 运行 `main.py` 启动工厂界面。
3. 在“智能去重”页选择素材目录。
4. 完成去重后，进入“交互标注”页对 Master 图片进行标注。
5. 标注完成后，在“模型训练”页一键启动训练并监控曲线。
