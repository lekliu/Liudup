# Liudup (六度图片去重) v2.7.2

**Liudup** 是一款专为 AI 数据集准备、广告素材清理及大规模图片库管理设计的专业去重工具。它结合了本地原生 AI 推理与 Minio 远程存储同步，通过视觉指纹技术精准识别并清理相似素材。

## 🛠 一、 核心架构复盘 (技术方案)

项目经历了从第三方框架 (`imagededup`, `FiftyOne`) 到**原生 AI 架构**的演进，目前采用以下最稳健的技术栈：

1.  **特征提取层**：原生 `torchvision` 加载 **MobileNetV2**。通过移除分类层，直接提取图片 1280 维的归一化特征向量。
2.  **高性能比对**：利用 `Numpy` 矩阵向量化运算实现余弦相似度比对，支持万级图片秒级对账。
3.  **持久化层**：采用 `SQLite3` 存储 `本地路径 <-> 远程 Key <-> 1280维向量 <-> 元数据` 的四位一体映射。
4.  **交互层**：`PyQt5` 打造。支持长截图优化的画廊展示及基于 `QScrollArea` 的高清缩放预览器。

## 🌟 二、 核心功能特性

### 1. 强一致性相似分组 (Strict Grouping)
*   **拒绝对称链式相似**：不同于传统的并查集算法（A像B，B像C $\rightarrow$ A,B,C一组），Liudup 采用**成员全校验逻辑**。
*   **逻辑要求**：新成员加入已有组前，必须与组内**所有**既有成员的相似度均达到阈值。这确保了每个相似组内的图片内容高度收敛。

### 2. 高性能增量分析 (Incremental Engine)
*   **断点续传**：已分析过的图片特征永久存入本地 `.db`，二次扫描耗时为 0。
*   **全量重算模式**：支持一键清空特征库，在更换模型或调整算法后重新扫描。
*   **失效清理**：分析前自动核对磁盘，若物理文件已删除，自动抹除数据库陈旧记录。

### 3. 专业级预览交互 (Pro Previewer)
*   **自适应窗口**：预览对话框根据图片比例动态定规，最大不超过屏幕尺寸。
*   **实时缩放**：支持鼠标滚轮对高清原图进行 10% ~ 1000% 的丝滑缩放，查看细节。
*   **组内导航**：预览状态下可通过键盘 `← / →` 键或 UI 按钮在相似组内快速切换。

### 4. 安全备份与同步 (Safety First)
*   **无损清理**：点击删除后，本地文件不会被抹除，而是移动到本地缓存根目录下的 `_backup` 临时备份区。
*   **云端联动**：同步删除 Minio 存储桶中的对应对象，保持云端数据纯净。

## 🚀 三、 快速开始

### 1. 环境准备
```bash
pip install torch torchvision numpy pillow PyQt5 boto3 python-dotenv
```

### 2. DLL 故障修复 (Windows 专用)
针对 Windows 环境下常见的 `OSError: [WinError 1114]`，Liudup 已在 `main.py` 顶层注入补丁，通过**主线程预加载 Torch** 的方式强制初始化 DLL。

### 3. 操作指引
1.  **设置路径**：选择本地图片存放目录（包含 Minio 下载的图片）。
2.  **同步远程**：拉取 Minio 桶内图片映射并补齐缺失文件。
3.  **开始分析**：
    *   调节 **“相似容差”** (建议 10-20)。
    *   点击 **“开始分析”**。
4.  **决策去重**：
    *   **双击**图片进入沉浸式对比模式。
    *   确认重复后，点击 **“彻底删除”** 将文件归档至备份区。

## 📊 四、 性能指标
*   **特征提取**：~10 张/秒 (Intel i5 CPU)。
*   **相似比对**：1000 张图片两两对账 < 0.1s。
*   **数据库容量**：每万张图片产生的 `.db` 文件约 50MB。
